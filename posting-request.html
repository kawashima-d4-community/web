Posting Request

地域情報掲載依頼

第四町内会・西谷・上星川周辺の地域情報をお寄せください。町内会側で内容を確認し、問題がなければホームページへ期間限定で掲載します。

メールで掲載依頼する 掲載中のお知らせを見る

掲載できる内容

  • 町内会・地域活動に関するお知らせ
  • 子ども会、防災、防犯、清掃活動の案内
  • 西谷・上星川周辺の地域イベント情報
  • 地域団体・関係機関からの公開情報
  • 町内会行事や地域活動の写真リンク

掲載できない内容

  • 個人情報を含む内容
  • 誹謗中傷、差別的表現、トラブルにつながる内容
  • 営利目的が強い宣伝
  • 宗教勧誘、政治的主張、公序良俗に反する内容
  • 掲載許可を得ていない人物写真

掲載依頼時に記載してほしい内容

投稿者情報氏名、所属・班名・団体名、連絡先メールアドレス
カテゴリ町内会、地域イベント、子ども会、防災・防犯、清掃活動、西谷・上星川情報、写真掲載など
掲載タイトル例：ラジオ体操のお知らせ、地域清掃ボランティア募集 など
本文日時、場所、対象者、内容、問い合わせ方法など
掲載希望期間原則1週間程度。イベント告知は開催日までなど、必要に応じて調整します。
リンク・写真Googleドライブ、公式サイト、案内ページなどのURL

掲載までの流れ

掲載依頼
メールで掲載したい内容を送付します。
内容確認
町内会側で内容・個人情報・写真許可を確認します。
期間限定で掲載
問題がなければホームページに掲載します。
期限後に非表示
掲載期限を過ぎた情報は画面に表示しません。

注意事項

すべての掲載依頼が掲載されるとは限りません。内容確認の結果、掲載を見送る場合や、表現を調整する場合があります。

写真に人物が写る場合は、投稿者の責任で掲載許可を得てください。特にお子様の写真は慎重に取り扱います。
