About Kawashima Daiyon Neighborhood Association

Kawashima Daiyon Neighborhood Association is a local community association for the area from 1 to 21 Higashi-Kawashima-cho, Hodogaya Ward, Yokohama.

Last updated: 2026-05-31

For first-time visitors

This association supports a safe and friendly neighborhood through disaster preparedness, crime prevention, cleaning activities, children’s activities, seasonal events, and local information sharing.

This website provides neighborhood announcements, event schedules, disaster-prevention information, and local community notices that can be checked on a smartphone.

Participation is welcome

Neighborhood events are for everyone in the community. You are welcome to join, watch, or help only when you can. Joining an event does not mean you will be asked to become an officer.

How to translate the whole website

The full website is mainly in Japanese. Please use Google Translate or the translation function in Chrome when you want to read other pages.

Open Japanese homepage Open Google Translate

On a computer using Chrome

  1. Open the Japanese homepage.
  2. Right-click on a blank area of the page and choose “Translate to English.”
  3. If English is not shown, click the translation icon in the address bar and choose your language.
  4. If the translation is not accurate, check the original Japanese page or ask a neighborhood officer.

On a smartphone using Chrome

  1. Open the Japanese homepage in Chrome.
  2. If a translation message appears, tap Translate.
  3. If it does not appear, tap the Chrome menu and choose Translate.
  4. Choose your language if needed.

Using Google Translate website

  1. Open Google Translate.
  2. Select “Websites.”
  3. Enter this URL: https://kawashima-d4-community.github.io/web/
  4. Choose the language you want and open the translated page.

Machine translation may not translate names of places, people, or events perfectly.
